Este é um trabalho da disciplina de Desenvolvimento de Software para Web, que é dividido em duas partes:
Entrega 1: HTML e CSS (versão com e sem ajuda de IA)
Entrega 2: HTML + CSS + JAVASCRIPT (versão com e sem ajuda de IA)

O projeto é sobre um site voltado para a extensão, chamado Portal de Saúde Comunitária de Quixadá, voltado para ajudar a população a se informar sobre 
Campanhas de Vacinação na cidade, Unidades de Saúde, Hábitos Saúdaveis entre outras diversas informações.
